# Makecode image gen
This project helps convert images into makecode java script image sprite code.
it takes inspiration from https://kristianpedersen.github.io/Convert-Image-to-MakeCode-Arcade-Sprite/

## Requirements
- website is hosted on github.io for free
- website has an upload image page
- once uploaded, user can select different resolutions and click generate to create the code
- code is easily copyable
